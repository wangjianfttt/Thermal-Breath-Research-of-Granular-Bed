## Official-format notes

Nature Physics currently states that initial submissions do not need special formatting and may be submitted as PDF, Word or TeX/LaTeX. For an Article, the current working envelope is a main text up to 3000 words, abstract up to 200 words and up to six main display items, with Methods and Extended Data handled separately. The local audit file `nature_physics_format_audit.csv` currently reports: abstract 191 words, main text excluding figures 2949 words, Methods summary 1318 words, five main figures and all five captions below 350 words.
